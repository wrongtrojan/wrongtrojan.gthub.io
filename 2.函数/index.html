November 24, 2025

函数

函数的定义与使用

函数定义

  返回值类型 函数名(形式参数){

  具体运算代码块；

  return 返回值;}

函数调用（函数的调用应该在声明或具体定义之后）

  嵌套调用

  递归调用

      直接或者间接的调用自身

      递推+回归

函数参数传递

  值传递

      将实参的值传递给形参

      函数内对形参操作不改变实参本身

  引用传递

      给实参起一个别名，引用给形参

      声明引用必须初始化指向已存在对象

      引用的底层实现是指针

  默认参数传递

      在声明形参类型时初始化

      若无相应实参传入则为默认值

  可变形参传递

      实参类型相同

          通过标准库类型initializer_list\<type\> list 生成一个列表对象

          initializer_list类提供的操作（服务）

              size( )元素数量

              begin( )返回首元素指针

              end( )返回尾元素下一位置指针

          以序列{}形式传入实参

      实参类型不同

          可变参数模板

内联函数（结构简单，语句少，调用频繁）

编译期间将函数体嵌入调用处

避免函数调用的栈帧开销（参数压栈、跳转、返回等）

语法形式 inline + 函数定义

constexpr函数

让编译器在编译阶段完成计算、内存分配操作，提升程序效率

用于常量表达式的函数，返回值、形式参数必须都为常量

必须有且仅有一条return语句

函数重载

函数名相同但形参个数/类型不同（无论返回值），编译器自动确定调用哪一个函数

使用默认形参值的函数重载时注意防止二义性

  void fun (int a,int b=0);

  void fun (int a);

系统函数

标准cpp函数（更强可移植性）

当前操作系统或编译环境中特有的

<cmath> sqrt,abs(绝对值),sin,cos,tan...
