类型判断

  auto类型

      让编译器替我们分析类型 定义时必须有初始值

      同样可以一次定义多个变量，但是变量间需类型一致

  decltype类型

      decltype（i）j 可以无需初始值，表达j与i类型一致

常量与变量

字面常量

  整型常量（八进制以0开头，十六进制以0x开头）

      后缀u表示unsigned 类型

  实型常量

      一般形式

      指数形式 e后数表示十的次方数

  字符常量

      单引号括起来

      ASCII码

  字符串常量

      每个字符占一节，在末尾添加\\0作为标记

  布尔常量

      false 和 true

变量

  变量声明 类型+名称

  变量初始化

      非列表初始化 "=""（）"

      列表初始化 "{}""={}"（不允许数据丢失）

  变量储存类型

      register 寄存器

      extern 所有函数和程序段中使用

      static 固定地址存放，整个运行期间有效

      thread_local 具有线程存储生存期

          修饰命名空间、块作用域中变量

          以及被指定为static存储类型的变量

      mutable 只作用于类数据成员

符号常量

  声明的同时初始化 const 类型 变量名=常量值

  程序中间不可改变其值

  constexpr关键字 验证是否为常量表达式，暗含const属性
